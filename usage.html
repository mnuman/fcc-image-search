Welcome to the Image Search API

This API has two distinct usages:
  1. searching for images
  2. displaying the last 10 searches performed

Example usages

Searching for images: https://cryptic-ridge-9197.herokuapp.com/api/imagesearch/lolcats%20funny?offset=10

Specify an offset=n parameter to skip the first n results and return n+1 to n+10

Retrieve the last queries: https://cryptic-ridge-9197.herokuapp.com/api/latest/imagesearch/
